Features

Alles was du brauchst

Von der Strategie-Analyse bis zur fertigen Präsentation — vollständig lokal.

🧠

Strategy Analysis

SWOT-Analyse und Porter's Five Forces via lokalem LLM + RAG-Kontext aus deiner Wissensdatenbank.

🔍

Deep Research

Aktive Web-Recherche (DuckDuckGo/Tavily) mit automatischem Ingest in die Qdrant Knowledge Base.

📊

Daten-Import

CSV und Excel-Dateien hochladen, Spalten erkennen lassen und automatisch als Chart-Slides einbetten.

💬

KI-Chat

Multi-Turn RAG-Chat im Editor. Frag nach, verfeinere Slides, erhalte kontextbezogene Vorschläge.

📈

Competitor Research

LLM-basiertes Scoring 1–10 für Wettbewerbsmatrizen. Automatisch befüllt aus Wissensdatenbank.

🎨

8 Layout-Typen

Agenda, KPI, SWOT, Timeline, Comparison, CTA, Image Focus — professionelle Layouts automatisch gewählt.

🔒

Vollständig Offline

STRATGEN_OFFLINE=true blockiert alle externen Calls. Kein Byte verlässt dein Netzwerk.

🧬

Self-Learning

Jeder Export und jedes Chat-Feedback wird in Qdrant indexiert. Das System lernt automatisch.

🖼️

Custom Templates

Eigene .pptx-Dateien als Basis-Template hochladen. Layouts und Farben werden übernommen.

Installation

In 5 Minuten starten

Voraussetzungen: Python 3.11+, Node.js 18+, Ollama mit mistral-Modell.

# 1. Repository klonen
git clone https://github.com/sodaen/stratgen.git
cd stratgen

# 2. Python-Backend einrichten
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. Frontend bauen
cd frontend && npm install && npm run build && cd ..

# 4. Konfiguration
cp .env.example .env
# .env anpassen: LLM_MODEL, QDRANT_URL etc.

# 5. Starten
gunicorn -k uvicorn.workers.UvicornWorker backend.api:app \
  --bind 127.0.0.1:8011 --workers 2
